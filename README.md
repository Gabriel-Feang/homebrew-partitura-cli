# Homebrew Tap for Partitura CLI

Official Homebrew tap for [Partitura CLI](https://partitura-ai.com) - AI-powered coding assistant.

## Installation

```bash
brew tap gabriel-feang/partitura-cli
brew install partitura-cli
```

## Usage

```bash
# Authenticate (opens browser)
partitura-cli auth

# Start chatting
partitura-cli

# Check status
partitura-cli whoami
```

## Updating

```bash
brew upgrade partitura-cli
```

## Uninstalling

```bash
brew uninstall partitura-cli
brew untap gabriel-feang/partitura-cli
```
